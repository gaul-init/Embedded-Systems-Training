Module 1: Project Management and Documentation

Topics:

  • Introduction to Git
  • Introduction to GitHub
  • Introduction to Web Publishing

Duration: 1 week

Projects: 1

Description

  1. The first week of the program covered basics of web publishing.
  2. We were introduced to Git and Github for version control and documentation.
  3. We also learned how to publish our projects on the web.
  4. We were given a project to create a personal website.
  5. The purpose of the website would be to track our progress in the program.
  6. The website would also serve as a portfolio for our projects.
  7. The project for the week was to host the website on Github pages.
  8. The website would be updated weekly with the progress of the program.
